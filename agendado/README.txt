Agendado

Aplicación que contiene el registro de contactos, utiliza SQL Lite.
Al iniciar la aplicación se genera la base de datos.

Modo de Uso
En el buscador ingresar el texto a buscar, se puede ingresar una sola letra. Se debe hacer click sobre el icono de búsqueda para que se realice la búsqueda.
Si no hay resultados encontrados la pantalla permanecerá vacía.
Hacer click sobre uno de los resultados para visualizar sus datos. Entre los datos se muestra el número de teléfono, para realizar llamado o mandar mensaje, correo de email y dependiendo del cargo a quienes reporta y personas a cargo.
